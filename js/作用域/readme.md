# 作用域
- 内层作用域能访问外层作用域的变量，而外层不能访问到内层私有的变量
    作用域从里往外找
- 尽量减少在全局作用域下的变量的定义

# 函数中的作用域 
- 函数会给自己创建一个作用域，其他的结构不会   (有特例，不完全正确)

## 变量声明 ，声明提升
- 如`var a = 1`的执行实际是：
    `var a`  该步为变量声明，声明会提升 如，若定义到全局，`var a`会提升到全局的最前方
    `a = 1`
 而es6中，let const 声明变量， 声明将不会提升, 解决了es5中的这个BUG
