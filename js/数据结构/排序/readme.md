# 希尔排序
  {55，26，33，80，70，90，6，30，40，20}，增量取5的希尔排序的第一趟的结果
  
  增量为5，即步长为5
  [0]55 [5]90  排序后为 [0]55 [5]90
  [1]26 [6]6   排序后为  [1]6   [6]26
  [2]33 [7]30 排序后为  [2]30 [7]33
  [3]80 [8]40 排序后为  [3]40 [8]80
  [4]70 [9]20 排序后为  [4]20 [9]70
  即55，6，30，40，20，90，26，33，80，70
