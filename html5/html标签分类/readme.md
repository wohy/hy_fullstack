# 第一种分类
1. 块集
    - 独占一行
    - 能设置宽高 
2. 行内
    - 都在一行上面
    - 不能设置宽高 (错误的)  应该说：行内非置换元素不能设置宽高，行内置换元素是可以设置宽高的
 img -> 行内 -> 但能设置宽高

# 第二种分类
1. 置换元素
    ```html
    <img src =  " " />   大小由src图片大小决定
    <canvas />     
    <input />
    <iframe src = ""/>
    <script src = ""/>
    ```
2. 非置换元素：
    ```html
    <div>
        <div></div>
    </div>
    <span></span>
    ```
